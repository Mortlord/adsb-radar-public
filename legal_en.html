What data is processed?

Location data (without Favorites feature)

The app requests browser geolocation permission once on launch. The location is used locally to centre the map and calculate distances to aircraft. It is not transmitted or stored.

Location data, favorites and Telegram Chat ID (with Favorites feature)

Users who activate the optional Favorites feature with Telegram notifications expressly consent to the following data being transmitted to and stored on the operator's server:

GPS-Standort Telegram Chat-ID Favoriten-Callsigns Konfigurierter Radius

Legal basis: Consent pursuant to Art. 6(1)(a) GDPR, given by actively saving data in the app. Data is used solely for delivering Telegram notifications and is not shared with third parties.

The server is operated via Railway (Railway Corp., San Francisco, USA). Data transfer to the USA is based on standard contractual clauses pursuant to Art. 46(2)(c) GDPR. Weitere Informationen: railway.app/legal/privacy.

Notifications are delivered via the Telegram Bot API (Telegram FZ LLC, Dubai, UAE). Weitere Informationen: telegram.org/privacy.

Stored data is automatically deleted after 30 days of inactivity. Immediate deletion is available at any time via the Favorites panel in the app.

Local browser storage (localStorage)

The app stores favorite callsigns and the Telegram Chat ID locally in the browser. This data is not automatically transmitted and can be deleted at any time via browser settings.

Requests to external services

When loading flight data, requests are made to the following third-party services:

airplanes.live api.adsbdb.com Timezone API

The user's IP address is technically transmitted to these services. The operator has no influence over their privacy practices.

What is not processed?

  • No cookies
  • No tracking or web analytics
  • No user accounts or registration
  • No sharing of data with third parties (except Telegram for notifications)

Hosting

This website is hosted via GitHub Pages (GitHub, Inc., San Francisco, USA). Server log data may be collected by GitHub when the page is accessed. Weitere Informationen: Datenschutzerklärung von GitHub.

User Rights

Users have the right to access, rectify and erase their stored data (Art. 15–17 GDPR). All stored data can be deleted directly via the Favorites panel in the app. For further questions: kontakt@adsb-radar.de.

Changes

This privacy policy will be updated when material changes are made to the app.
